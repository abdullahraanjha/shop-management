# Future Improvements

- **Weighted-average costing** — option to use moving-average cost instead of
  latest-cost, for businesses that prefer it.
- **Multi-store / warehouses** — per-location stock and transfers.
- **Role-based permissions UI** — fine-grained access control screens.
- **Returns & refunds** — sales returns (stock back in) and purchase returns.
- **Barcode scanning** — hardware scanner support at the point of sale.
- **Receipt/thermal printing** — 58mm/80mm receipt templates.
- **Payment tracking** — partial payments, due dates, customer/supplier ledgers.
- **Export** — CSV/Excel/PDF export for every report.
- **Audit log** — who changed what and when across all entities.
- **Testing** — unit tests for services, integration tests for the stock/profit
  transactions, and end-to-end tests for critical flows.
- **CI/CD & Docker Compose** — one-command spin-up (app + PostgreSQL) and
  automated checks on every push.
- **Internationalization** — multi-currency and multi-language support.
